The Problem

FizzBuzz (But Make It Weird)

Write a program that prints numbers from 1 to 100. But:

  • For multiples of 3, print "Fizz"
  • For multiples of 5, print "Buzz"
  • For multiples of both 3 and 5, print "FizzBuzz"
  • Otherwise, print the number

The twist: You must pick one of the constraints below and follow it religiously. The more creative your interpretation, the better!

Starter Code (feel free to ignore):

// Your constrained masterpiece goes here
// Remember: there's no wrong way to do this!

function fizzBuzz() {
    // TODO: Implement with your chosen constraint
}

fizzBuzz();
Pro tip: The goal isn't to write good code. It's to have fun with the constraint and maybe learn something weird!

Example Solutions

Gordon Ramsay Mode:

function thisNumberIsBloodyRaw() {
    for (let ingredient = 1; ingredient <= 100; ingredient++) {
        let dish = "";
        if (ingredient % 3 === 0) dish += "Fizz";
        if (ingredient % 5 === 0) dish += "Buzz";
        
        if (dish === "") {
            console.log(`ORDER ${ingredient}: IT'S RAW!`);
        } else {
            console.log(`${dish}... FINALLY, SOME GOOD FOOD`);
        }
    }
}

No Loops Allowed:

function fizzBuzzNoLoops(n = 1) {
    if (n > 100) return;
    
    const output = n % 15 === 0 ? "FizzBuzz" :
                   n % 3 === 0 ? "Fizz" :
                   n % 5 === 0 ? "Buzz" : n;
    
    console.log(output);
    fizzBuzzNoLoops(n + 1);
}
